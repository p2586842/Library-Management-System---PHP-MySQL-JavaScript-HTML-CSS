This project is a prototype Library Management System that allows users to interact with a digital collection of books. It features:

User Roles: Admins can manage the book inventory, while users can borrow and return books.
Database-Driven: All data is stored and retrieved from a MySQL database.
Security Features: Includes input validation and password hashing for secure user authentication.
Accessibility: Designed with accessibility in mind for ease of use.
Scalability: Modular code structure allows for easy future enhancements.
Key Technologies:

Frontend: HTML, CSS, JavaScript
Backend: PHP (no frameworks)
Database: MySQL/MariaDB
How to Run:

Install AMPPS/WAMP/XAMPP or any similar stack.
Clone the repository to the www directory.
Import the provided SQL file into your MySQL database.
Access the system via http://localhost/library_system/
